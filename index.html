Unix-Natural Language Timestamp Microservice

This api provides a json object with both unix and a natural time string from a unix or natrual time string. Project for Free Code Camp by Andrew VanVlack

Example Inputs

  1. http://timestamp.com/March%205,$201989
  2. http://timestamp.com/605088000

Example Output

{
	unix: 605088000,
	natural: "March 5, 1989"
}

Node.js source code available on github
